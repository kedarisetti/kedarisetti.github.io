About Me

I am Dharanish Kedarisetti, based in Boston. Applied ML engineer with 5+ years building production models and edge systems — currently a Research Software Engineer at Amazon working on smart cart technology.

Interests: Machine Learning, Climbing, Meditation, and Photography.

Experience

Motorola Solutions — Sr. Machine Learning Engineer, Somerville, MA (Jan 2020–Sept 2022) Smart camera features for context similarity, unusual activity detection, and crowd analysis. CNN model deployment via ONNX, Docker, and Kubernetes. Contributed to the patent committee.

Avigilon — Research Engineer / Software Engineer, Somerville, MA (Feb 2018–Jan 2020) Video analytics framework development focused on unusual activity detection in C++ and Python.

Patents

  • US11303877B2 — Method and System for Enhancing Use of Two-Dimensional Video Analytics by Using Depth Data (Avigilon Corp, granted 2022). Improves 2D video analytics by fusing depth data to validate object classifications, correcting for shadows, reflections, and distance-based size distortions. With Pietro Russo, Peter L. Venetianer, Mahesh Saptharishi.

  • US20210350117A1 — Anomalous Pose Detection Method and System (Motorola Solutions, 2021). Uses neural networks to generate skeletal pose vectors and flags deviations from learned normal-behavior clusters in real time. With Teresa Dankel, Patrick Henson, Aral Sarrafi.

  • WO2020210504A1 / US11302117B2 — Anomaly Detection Method, System and Computer Readable Medium (Avigilon Corp / Motorola Solutions, granted 2022). Detects unusual behavior in surveillance video by modeling statistical baselines from object metadata (class, location, velocity) per scene cell — without processing raw frames. With Peter Venetianer.

  • WO2020051704A1 — System and Method for Improving Speed of Similarity Based Searches (Avigilon Corp, 2020). Accelerates appearance-search queries by k-medoid clustering of selected feature vectors before database search, reducing compute without sacrificing result quality. With Nicholas John Alcock, Peter L. Venetianer.

  • US12634398 — Method of Generating a Trigger Initiating an Emergency Incident Response Workflow (Motorola Solutions, 2026). Receives emergency notification data and retrieves associated sensor data by time and location to automatically initiate incident response. With Emmy Beltre, Kaveh Malakuti, Mariya Bondareva.

Papers

  • Large-scale Indoor Mapping with Failure Detection and Recovery in SLAM — Sharmin Rahman, Rob DiPietro, Dharanish Kedarisetti, Vinod Kulathumani. IROS 2024. Proposes an automated visual-inertial SLAM service with a health tracker for faulty measurements and continuous session merging, validated in grocery stores spanning 1700–3700 m². [PDF] [Amazon Science]

  • Plus Disease in Retinopathy of Prematurity: Improving Diagnosis by Ranking Disease Severity and Using Quantitative Image Analysis — Jayashree Kalpathy-Cramer, J Peter Campbell, Deniz Erdogmus, Peng Tian, Dharanish Kedarisetti, et al. Ophthalmology 123(11):2345–2351, 2016. Shows that while experts disagree on categorical ROP diagnosis, pairwise severity ranking yields good inter-expert correlation (mean r = 0.84), and that computer-based image analysis can model this continuous spectrum. [DOI] [PubMed]
